# Contract: agent files (`agents/<name>/`)

Per-run flat files, inside the save directory, travel with `cp -R` (FR-011).

```text
<world>/agents/
├── ash/
│   ├── persona.md   # authored at genesis, mode 0444, never written again
│   └── soul.md      # regenerated by the scribe while the daemon runs
├── birch/ …         # one dir per agent, lowercase name
```

## persona.md (immutable)

Authored constants in `internal/persona`; written exactly once by `promptworld new`.

```markdown
# Ash

**Temperament:** steady, practical, slow to anger.
**Drives:** keep everyone fed; distrusts idleness.
**Quirk:** talks to the fire as if it answers.
**Bonds:** protective of Fern; old rivalry with Oak.
```

Rules: mode `0444`; no code path writes it post-genesis (the firewall's structural
half); the mind reads it as the stable prompt prefix. Content-drift validation is
TASK-9's half.

## soul.md (regenerated view)

Source of truth is the event log (`agent.memory_added` reduced into state); the file
is a render, safe to delete (regenerates), safe to read live.

```markdown
# Ash — soul

*Born day 1. Alive. 14 memories.*

- **day 1 06:52** (5★) Built a fire on the plain.
- **day 1 22:04** (5★) Survived a freezing night by the fire.
- **day 2 09:31** (3★) Talked with Birch.
- **day 3 14:02** (9★) Nearly died — starving and far from food.
```

Dead agents get a final line (`*Died day N — <cause>.*`) and the file freezes.
