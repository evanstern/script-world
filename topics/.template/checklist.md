# Understanding Checklist: <Topic / Lesson>

> Copy this folder to `topics/<topic-slug>/<NNN>-<lesson-slug>/` at the **start** of a lesson.
> Check items off only when understanding is demonstrated (in your own words / correct reasoning), not merely acknowledged.

## 1. The Problem
- [ ] What problem does this solve?
- [ ] Why did this problem exist in the first place?
- [ ] What are the different ways it could have been solved?

## 2. The Solution
- [ ] How does it work?
- [ ] Why was it designed this way?
- [ ] Key design decisions and tradeoffs
- [ ] Edge cases and failure modes

## 3. The Bigger Picture
- [ ] Why does this matter?
- [ ] What does it affect or enable?
- [ ] How does it fit into the broader context?
